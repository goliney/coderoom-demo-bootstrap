# coderoom-demo-bootstrap
> [Coderoom](https://github.com/goliney/coderoom) demo for [Bootstrap](http://getbootstrap.com/) snippets

## Demo
http://goliney.github.io/coderoom-demo-bootstrap/

## How to build
```sh
npm install
npm run build
```

If coderoom is installed globally:

```sh
coderoom demo/ build/
```

Or via grunt:

```sh
npm install -g grunt
npm install
grunt
```
